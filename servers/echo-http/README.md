# Echo HTTP MCP Server

Streamable HTTP transport를 사용하는 테스트 MCP 서버.

## 로컬 실행

```bash
npm install
npm start
```

서버가 `http://localhost:8080/mcp` 에서 실행됩니다.

## 도구

- **echo** — 입력 메시지를 그대로 반환
- **greet** — 이름으로 인사말 생성 (한/영/일)
- **timestamp** — 현재 서버 시간 반환

## MCP Zero에 등록

MCP 서버 URL: `http://localhost:8080/mcp` (로컬) 또는 배포된 URL
